    如何链式提示 #

    1. 识别子任务：将任务分解为不同的、连续的步骤。
    2. 使用XML构建清晰的交接：使用XML标签在提示之间传递输出。
    3. 设定单一任务目标：每个子任务应该有一个明确的单一目标。
    4. 迭代：根据Claude的表现改进子任务。
    example

    事例：分析法律合同
