    Query Transformation #

    Multi Query多查询策略[3] #

    该方法从多个角度重写用户问题，为每个重写的问题检索文档，返回所有查询的唯一文档。

    Decomposition问题分解策略[3] #

    • Answer recursively迭代式回答 在问题分解的基础上，逐步迭代出答案，将上一步问题的答案，与下一步骤的答案进行拼接，送入大模型进行问答

    • Answer individually 也可以让每个subquery分别进行处理，然后得到答案，然后再拼接成一个QA pairspprompt最终形成答案。
